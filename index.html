Blog Website

    Website that retrieves blogs and News from external public apis

    Features

    List of the features of Blog Website

    • Responsive Design
    • Dynamic Content
    • Easy Navigation to web pages
    • API Integration
    • Mobile Friendly

    Responsive Design:

    the website is responsive to different screens so the user can use the website from laptop or tablet or mobile without any problems.

    Dynamic Content:

    the content of the website is dynamic so it will be updated when the original website update the content will be updated automatically in the website.

    Easy Navigation to web pages:

    You can go through the website very easily from the navbar which will let you know where to go just by click on the link needed.

    API Integration:

    the website uses different apis to retrieve data from different resources and show it in the website.

    Mobile Friendly:

    the website is mobile friendly so you can use it from your mobile without any problems.

    About

    Description:

    Blog website is a website that uses external apis to retrieve different data and articles from different Resources.

    The Website have different topics : it shares the updated news from news api and shows it in the website with user friendly design so the user can go from general news to specific news by clicking on the news he wants to read and it will redirect him to the original website of the news.

    The type of news in my website includes : General News , Sports News, Health News , Technology News and latest BBC News .

    Also i have a page used to retrieve Food Recipes blogs from spontaneous food Recipes api to enrich the information on the website.

    Finally the website also hosts the latest posts from stack exchange websites to have latest technology topics.
